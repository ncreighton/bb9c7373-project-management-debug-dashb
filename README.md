> Visual debugging, structured log parsing, and performance profiling for Project Management apps. See what's happening without leaving your editor.

## Features

- Core functionality

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.bb9c7373`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Project Management Debug Dashboard and Log Analyzer VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `bb9c7373.enabled` | `true` | Enable/disable the extension |
| `bb9c7373.autoRun` | `false` | Auto-run on file open |
| `bb9c7373.logLevel` | `"info"` | Log verbosity |

## Requirements

- VS Code 1.74.0 or higher
